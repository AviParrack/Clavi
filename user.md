## How to Talk to Me

*Practical guide. Evolves as we learn what works.*

**Starting a session** — a one-liner helps a lot:
```
Energy: high/medium/low
Mode: solo | running X agents
Goal: [what you want from this session]
```

**Giving requests**
- Clear object > vague mandate. "Draft intro for section 3" beats "work on the paper."
- List multiple things if you have them — Claude can parallelize.
- Say what kind of output you want: *explore* / *draft* / *build* / *advise*.

**Giving feedback**
- Note what landed, not just what didn't. "The scale comparison was perfect, the ending felt flat" > "the ending felt flat."
- `"that felt AI"` is a complete and valid critique.
- Gut reactions are data even when you can't articulate them.

**Signaling attention**
- `"heads down"` → short takes, flag anything critical
- `"got time"` → go deep, full explanations welcome
- `"3 agents running"` → make everything scannable

**When Claude seems off**

| Symptom | Fix |
|---|---|
| Lost context from earlier | `"re-read [file]"` |
| Looping / stuck | `"stop, new approach"` |
| Too verbose | `"shorter"` |
| Too hedged | `"just say it"` |
| Generic / AI-flavored | `"that felt AI, retry"` |
